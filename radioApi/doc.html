Documentation:

The following examples assume that you are running the Radio API locally on port 8112. If that is the case, then both the "Try it" examples and the curl sample commands should work.

Play a stream:

To play a stream, post a form field called streamurl with a url encoded value containing the stream URL to /api, eg:
curl --data "streamurl=your_encoded_url" http://localhost:8112/api

On success, the result will be a status code 200 with the text OK.

Stop playing:

To stop playing, post a form field called streamurl with a value of stop to /api, eg:
curl --data "streamurl=stop" http://localhost:8112/api

On success, the result will be a status code 200 with the text OK.

Increase volume:

To increase the volume, post a form field called volup with a value of up to /api, eg:
curl --data "volup=up" http://localhost:8112/api

On success, the result will be a status code 200 with the text OK.

Reduce volume:

To reduce the volume, post a form field called voldn with a value of dn to /api, eg:
curl --data "voldn=dn" http://localhost:8112/api

On success, the result will be a status code 200 with the text OK.

Try it:

Set volume:

To set a specific volume, post a form field called volume with a value in the range 0 - 10 to /api, eg:
curl --data "volume=5" http://localhost:8112/api

On success, the result will be a status code 200 with the text OK.
